# lofi-stream-kick

24/7 lofi beats streamed to Kick with an arcade/retro gaming aesthetic.

## Live Stream

Watch: [Coming Soon on Kick]

## Theme

Arcade/retro gaming vibes with:
- Neon-lit arcade cabinet with CRT monitor
- Animated pixel art visualizer
- Chiptune-style lofi beats (8-bit aesthetic)
- "INSERT COIN" neon sign

## Local Development

```bash
cd docs
python3 -m http.server 8080
# Open http://localhost:8080
```

## Tech Stack

- HTML/CSS for visuals
- Web Audio API for generative chiptune audio
- GitHub Pages for static hosting
- Xvfb + Chromium + FFmpeg for streaming
- systemd for auto-recovery
